# AlgoReminder
My Simple app for Algorithm Learning
## Requirements
MacOS >=15.0
## Version
0.720
## Features
1.已经实现能够导入题目信息形成题目卡片，导入MarkDown格式笔记形成笔记卡片 

2.根据艾宾浩斯浩斯遗忘曲线智能调度复习进度，每日定时提醒

3.笔记支持关联题目，笔记支持app内查看功能
## Next Version
1.支持笔记在线编辑功能

2.支持自定义设置复习频率与提醒时间

3.题目查看题面和题解等信息
